# Fingerspelling Recogniotion using [MediaPipe](https://github.com/google/Mediapipe) on the Web
[MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands)を用いて手の姿勢推定を行い、
取得した手の形状パラメータを用いて機械学習で、かな指文字を分類するプログラムです。  
こちらは、[Fingerspelling Recognition using MediaPipe](https://github.com/redgum775/fingerspelling-recognition-using-mediapipe)のWeb版です。

## Reference  
- [google/Mediapipe](https://github.com/google/Mediapipe)  
- [Kazuhito00/hand-gesture-recognition-using-mediapipe](https://github.com/Kazuhito00/hand-gesture-recognition-using-mediapipe)（分類モデルの構築に参考させて頂きました）  
- [Fingerspelling Recognition using MediaPipe](https://github.com/redgum775/fingerspelling-recognition-using-mediapipe)（Python版）
