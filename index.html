Fingerspelling Recogniotion using Mediapipe on the Web

「あ」～「ん」までの50音の指文字に対応しています。濁音、半濁音、拗音は対応していません。（2022.09.27 時点）


認識結果: _


MediaPipe Handsを用いて手の姿勢推定を行い、取得した手の形状パラメータを用いて機械学習により、かな指文字を分類するプログラムです。また、こちらはFingerspelling Recognition using MediaPipeのWeb版です。TensorFlow.jsでTensorflow Liteの指文字分類モデルを動かしています。

指文字とは

指文字は手の形状を「あ」～「ん」の50音に対応せた視覚言語です。主に聴覚障がい者が、手話と併せてコミュニケーションで使います。

指文字表現を知らない方は、指文字一覧｜NHK手話CGで指文字表現を確認できます。併せてご利用ください。
